
Sección 1.16
1.1 Liste 3 ejemplos de objetos reales que no mencionamos. Para cada objeto liste varios atributos y comportamientos.
	1 Alimentos
		Atributos: Color, olor, sabor.
		Comportamientos: Se cocinan, se comen, se dañan.
	2 Libros
		Atributos: Páginas, autor, secciones.
		Comportamiento: leíbles, se compran, se guardan.
	3 Animales
		Atributos: Especie, tamaño, alimentación.
		Comportamiento: Crecen, juegan, comen.
	
1.2 El pseudocódigo es _un medio informal para expresar la lógica de un programa_.

1.3 El UML se utiliza principalmente para _diseñar sistemas orientados a objetos_.
